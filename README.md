### **[UAV Camera Gimbal Controller](https://github.com/LeehamHunt/UAV-Gimbal-Controller)** - Embedded Control System for stabilized UAV imaging payload

C/C++, PlatformIO, Arduino/SAMD21, MAVLink, CAN Bus, SimpleFOC
